  	\Sigma; \Gamma \vdash e_0 : T_0 \\

  	\Sigma; \Gamma \vdash S' <: S \dashv \Gamma \\
  	\Sigma; \Gamma \vdash T <: U \dashv \Gamma}
  {	\Sigma; \Gamma \vdash e_0.m_U(e_1) : U}
